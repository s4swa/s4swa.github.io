What is

Discord.js is what is known as a library. an API's library. Now first of all what's an API? Well API stands for "Application programming interface". What it does is that it allows users to interact with a system and use various functions. A library is a program used to connect to that API. Since Discord doesn't have an API in a specific programming language, a library allows you to use normal code to connect to the API in the programming language the library is written in. Discord.js is written in javascript and allows the user to connect to the public API in javascript. Discord.js is used to make bots or bot accountsthat respond to various comands typed into the chat and act accordingly. For example one could make a command that if someone writes "ping" it replies "pong".
